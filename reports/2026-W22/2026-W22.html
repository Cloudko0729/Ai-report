1. Google Antigravity 2.0 — Agent-First 開發平台重磅登場（Google I/O, 5/19）

Google 發布 Antigravity 2.0，五大元件完整亮相：Desktop App、CLI、SDK、Managed Agents API 以及 Enterprise Agent Platform，底層驅動模型為 Gemini 3.5 Flash。新版核心設計是 Agent Teams 面板：可組合多個 subagent、各自在獨立 sandbox 並行執行，統一 timeline 追蹤所有 agent 事件，並支援背景排程任務。

主秀展示：Varun Mohan 用 Antigravity 2.0 的平行 agent 陣列，以不到 $1,000 token 成本從零建出一個可執行的 OS kernel。

定價調整： AI Ultra $100（比 Pro 多 5x 用量）；AI Ultra 頂規從 $250 降至 $200（20x 用量）。

  • Gemini CLI 強制遷移：6/18 停止服務，僅給 30 天通知；GitHub 討論串 24 小時內收到 143 個 thumbs-down vs 4 個 cheers
  • 閉源化：Gemini CLI 本來開源，Antigravity CLI 在發布時為閉源
  • UI 自動更新事故：2.0 rollout 將 code editor 介面替換成 chat 介面，不少用戶的開發環境當場失能
Antigravity 2.0 是 Claude Code 的直接競爭者，Agent Teams 的並行 subagent 設計模式值得深入研究；若有在用 Gemini CLI，6/18 前需完成遷移。

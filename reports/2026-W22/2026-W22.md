## 本週最值得注意

### 1. Google Antigravity 2.0 — Agent-First 開發平台重磅登場

Google 於 I/O 2026（5/19）發布 Antigravity 2.0，五大元件完整亮相：**Desktop App**、**CLI**、**SDK**、**Managed Agents API** 以及 **Enterprise Agent Platform**，底層驅動模型為 Gemini 3.5 Flash。新版核心設計是 **Agent Teams 面板**：可組合多個 subagent、各自在獨立 sandbox 並行執行，統一 timeline 追蹤所有 agent 事件，並支援背景排程任務。

主秀展示：Varun Mohan 用 Antigravity 2.0 的平行 agent 陣列，以不到 $1,000 token 成本從零建出一個可執行的 OS kernel。

**定價調整：** AI Ultra $100（比 Pro 多 5x 用量）；AI Ultra 頂規從 $250 降至 $200（20x 用量）。

**爭議四起：**
- **Gemini CLI 強制遷移**：停止服務，僅給 30 天通知，GitHub 討論串 24 小時內收到 143 個 thumbs-down vs 4 個 cheers
- **閉源化**：Gemini CLI 本來開源，Antigravity CLI 在發布時為閉源
- **UI 自動更新事故**：2.0 的自動 rollout 將 code editor 介面替換成 chat 介面，不少用戶的開發環境當場失能

**對你的意義：** Antigravity 2.0 是 Claude Code 的直接競爭者，Agent Teams 的並行 subagent 設計模式值得深入研究；若有在用 Gemini CLI，需完成遷移。

### 2. Gemini 3.5 Flash — Flash 級打敗 Pro 的里程碑

同樣於 I/O 2026（5/19）發布，Gemini 3.5 Flash 在工具協調基準上全面超越上代 Pro：

| 基準 | Gemini 3.5 Flash |
|------|-----------------|
| Terminal-Bench 2.1 | 76.2% |
| MCP Atlas（工具協調） | **83.6%**（AI coding 工具目前最高） |
| CharXiv Reasoning（多模態） | 84.2% |
| 輸出速度 | 比同級 frontier 模型快 4x |

**規格：** 1M context window、64K output；**定價：** $1.50 input / $9.00 output per 1M tokens，**$0.15 cached input**。Google AI Studio 與 Gemini API 即日起可用。

**對你的意義：** MCP Atlas 83.6% 意味著 tool-heavy agentic workflow 目前的最強方案之一；$0.15 cached input 對 system prompt 重複率高的場景有顯著成本優勢。

### 3. Cursor Composer 2.5 — Kimi K2.5 平價追齊 Opus 4.7

Cursor 於 5/18 發布 Composer 2.5，基礎架構是 Moonshot AI 開源的 **Kimi K2.5**（MoE，1 兆總參數、約 32B 活躍參數），三大訓練改進：

- **25x 更多合成訓練任務**（含「功能刪除後重建」難題）
- **Textual Feedback RL**：在每次工具呼叫失敗時給局部提示，非只有最終 reward
- **MoE 規模基礎設施**：sharded Muon optimizers、dual-mesh HSDP

**基準比較：**

| 基準 | Composer 2.5 | Claude Opus 4.7 |
|------|-------------|-----------------|
| SWE-Bench Multilingual | 79.8% | 80.5% |
| CursorBench v3.1 | **63.2%** | 61.6% |
| Terminal-Bench 2.0 | 69.3% | 69.4% |

**定價：** Standard $0.50/$2.50、Fast $3.00/$15.00 per 1M tokens — **約為 Opus 4.7 的 1/10**。

**對你的意義：** 效能幾乎追齊 Opus 4.7 且成本只有 1/10，MoE + 大規模合成資料訓練路線正在追上 frontier closed model。

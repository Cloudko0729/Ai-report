## 本週最值得注意

### 1. Gemini 3.5 Pro 第三度跳票：07-17 目標又落空，DeepMind 打掉重練基礎模型

本週最大轉折：Google 傳聞的 **07-17 Gemini 3.5 Pro GA 日期又沒兌現**，成為第三次延期。

**事件經過：**
- 週初洩漏資訊指向 07-17 GA（200 萬 token context、Deep Think 推理、$1.25/$10 定價）
- **Bloomberg 07-16 報導**：發佈再次延後，模型未達 Google 自訂的品質門檻——問題出在**幻覺率**與真實場景可靠度
- DeepMind 據報**打掉原始基礎模型重練**：工程師發現兩個結構性失敗模式——複雜多層 SVG 場景佈局無法維持結構一致性、遞迴式 tool-calling 環境下會崩潰
- 截至 07-18，Gemini 3.5 Pro 仍未出現在 Google 公開 API 模型清單，官方 help 頁仍只描述 Flash

**對你的意義：** 上週報告把 07-17 GA 列為觀察重點，現在答案揭曉：沒發生，而且是打掉重練等級的延期。三大廠中 Google 落後的幅度比預期更大；下週不宜再預設 GA 日期，改為觀望 DeepMind 重建後的實際成果。

### 2. OpenAI 提議把 5% 股份交給美國政府（估值約 $852B，對應約 $42.6B）

延續上半週的 IPO 傳聞，OpenAI 本週再出一記震撼彈：**Sam Altman 直接向 Trump 總統、商務部長 Lutnick、財政部長 Bessent 提議，把公司約 5% 股份交給美國政府**——以近期 $8520 億估值計算，價值約 **$426 億美元**。

**對你的意義：** 這是 AI 巨頭與政府關係前所未見的操作，牽動 OpenAI 出口管制待遇、政府採購、監管立場；與上半週的機密 IPO 籌備（目標 9 月上市）疊在一起看，OpenAI 正在同時佈局資本市場與政治靠攏。

### 3. Anthropic 本週連發：Claude for Teachers、自助 HIPAA、對話中系統訊息、記憶升級

- **Claude for Teachers（07-14）**：免費提供全美 K-12 教師使用，含教學技能庫與對應全 50 州學術標準、實證課綱的直接連結。加入 Google、OpenAI、Khan Academy 搶佔教室 AI 的戰局。
- **自助 HIPAA 設定（07-15）**：Enterprise / API 組織可自助審閱 BAA、下載實作指南、一鍵啟用 HIPAA 設定，醫療場景落地門檻大幅降低。
- **對話中系統訊息（mid-conversation system messages，07-15~16）**：Claude API、Bedrock、Google Cloud 上線，支援 Fable 5 / Mythos 5 / Opus 4.8——可在對話進行到一半插入系統指令，對長 agent workflow 的動態控制很有用。
- **記憶升級（07-10 上線，07-16 報導）**：以個別分類條目取代原本的每日記憶摘要，提供更豐富的對話脈絡。

**對你的意義：** 「對話中系統訊息」對你跑週報這類多階段 pipeline 特別有價值——可以在中途調整 agent 行為而不必重開對話。記憶改成分類條目後，跨對話的脈絡連貫性會更好。

### 4. 🔻 Claude Code 週用量 +50% 加碼已到期（2026-07-13 18:00 PT）

上半週最緊急的項目已成過去式：Anthropic 自 05-13 起提供的 **50% 週用量加碼，已於 07-13 美西時間 18:00 到期**，週用量回復到加碼前水準。這是繼 5/6 五小時額度翻倍、6 月移除尖峰限制之後的一連串「反 Codex」加碼的收尾——背景是 Codex 同等任務 token 消耗量僅約 Claude Code 的 1/4.2。

**對你的意義：** 如果你這週感覺 Claude Code 週額度比上週緊，就是這個原因。目前沒有續期公告，規劃高用量任務時要以回復後的額度為準。

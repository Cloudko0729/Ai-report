🔥 本週最值得注意

1. 🔴 Claude Code 週用量 +50% 加碼「今天到期」（2026-07-13 PT 18:00）

Anthropic 從 2026-05-13 起給 Pro / Max / Team / seat-based Enterprise 用戶加碼 50% 週用量上限，這是繼 5/6 五小時額度翻倍、6 月移除尖峰時段限制之後，5 週內第三次針對可用量的調整——背景是 OpenAI Codex 同等任務 token 消耗量僅約 Claude Code 的 1/4.2，Anthropic 用這一連串加碼正面迎戰。

這波加碼將於今天美西時間 18:00 到期，之後回復原本額度。

如果這幾天有大型 refactor 或高用量任務，建議在額度回復前（今天內）盡量跑完；到期後週用量會縮回加碼前水準。

2. OpenAI 機密送件 IPO：目標 2026 年 9 月上市，估值約 $730B

OpenAI 正與高盛、摩根士丹利準備機密 IPO 申請，目標最快 2026 年 9 月公開上市，私募市場估值約 $7300 億美元——若成真將是史上規模最大的科技業 IPO。

若你有追蹤 AI 產業的資本動態，這是本季最大的結構性訊號；IPO 前的靜默期可能影響 OpenAI 未來幾個月的產品發佈節奏與公關口徑。

3. Apple 控告 OpenAI 竊取商業機密（2026-07-11）

Apple 在北加州聯邦法院對 OpenAI 提告，指控竊取商業機密，並指出目前有 400 多名前 Apple 員工在 OpenAI 任職，多數來自 Apple 晶片設計、硬體、裝置端 AI 團隊。同日，OpenAI 安全主管 Heidecke 在一波人事改組後離職。

首起大型科技公司對 OpenAI 提起的商業機密訴訟，若進入實質審理，可能牽動 OpenAI 的裝置端 AI（硬體）野心與相關人才佈局。

4. OpenAI 關閉 Atlas 瀏覽器，功能併入 ChatGPT 桌面版與 Chrome 擴充套件

OpenAI 宣布 sunset 其 2025-10 推出的 Atlas AI 瀏覽器，預計 2026-08-09 停用，上線不到 9 個月。取而代之：新的 ChatGPT Chrome 擴充套件（讀取頁面脈絡、回答頁面問題、啟動長任務）與 ChatGPT 桌面版強化瀏覽能力（瀏覽網站、登入帳號、下載檔案）。

背景：OpenAI 應用程式前 CEO Fidji Simo 先前要求團隊減少「side quest」，此前已關閉 Sora 影片生成工具，Atlas 是同一波收斂策略下的犧牲品。

功能沒有消失，只是重新分佈到 Chrome 擴充套件與桌面版；獨立瀏覽器產品線收斂，反映 OpenAI 正把資源集中到 agentic browsing 的核心能力而非獨立產品外殼。

5. Google Search 結果頁全面由 Gemini 3.5 Flash 生成（2026-07-10）

截至 2026-07-10，Google 搜尋結果頁已全面由 Gemini 3.5 Flash 生成答案，取代傳統十條藍色連結列表；來源引用改成嵌入在 AI 生成答案內文中，而非列在答案下方。

這是搜尋引擎史上最大的介面轉變之一，SEO/內容策略需要重新思考「被引用在 AI 答案內」而非「排名第一」。

6. Gemini 3.5 Pro 洩漏資訊：預告 07-17 GA，200 萬 token context，Deep Think 推理模式

多方洩漏資訊指向 Google 將於 2026-07-17 讓 Gemini 3.5 Pro GA：

  • 200 萬 token context window
  • Deep Think 推理模式，僅限 $250/月 Ultra 訂閱
  • API 定價約 $1.25 input / $10 output per 1M tokens
  • 洩漏測試結果分歧：一說前端設計上超越 Fable 5，另一說推理與程式碼上仍落後 Fable 5 與 GPT-5.6
若洩漏屬實，下週很可能是 Gemini 3.5 Pro 正式 GA 週，屆時可以做三大廠模型的完整定價/能力對比。

7. Claude Code 本週功能更新（2026-07-11）

  • Auto mode 在 Bedrock / Vertex AI / Foundry 預設開啟
  • Bedrock 更新至 Claude Opus 4.8
  • Chrome 整合正式脫離 preview，開放給直接向 Anthropic 訂閱的所有用戶
  • Claude Desktop 登陸 Ubuntu 22.04+ / Debian 12+（beta）：x86_64 與 arm64 皆支援
  • Agent view 與 Remote Control 改善，修正凍結、當機、憑證問題與多個 worktree/plugin bug
  • Fable 5 免費存取期限延長至 07-12 23:59:59 PT（本報告產出時已過期）

🔥 本週最值得注意

1. Gemini 3.5 Pro 第三度跳票：07-17 目標又落空，DeepMind 打掉重練基礎模型

本週最大轉折：Google 傳聞的 07-17 Gemini 3.5 Pro GA 日期又沒兌現，成為第三次延期。

  • 週初洩漏資訊指向 07-17 GA（200 萬 token context、Deep Think 推理、$1.25/$10 定價）
  • Bloomberg 07-16 報導：發佈再次延後，模型未達 Google 自訂的品質門檻——問題出在幻覺率與真實場景可靠度
  • DeepMind 據報打掉原始基礎模型重練：複雜多層 SVG 場景佈局無法維持結構一致性、遞迴式 tool-calling 環境下會崩潰
  • 截至 07-18，Gemini 3.5 Pro 仍未出現在 Google 公開 API 模型清單
上週報告把 07-17 GA 列為觀察重點，現在答案揭曉：沒發生，而且是打掉重練等級的延期。三大廠中 Google 落後幅度比預期更大；下週不宜再預設 GA 日期，改為觀望重建後的實際成果。

2. OpenAI 提議把 5% 股份交給美國政府（估值約 $852B，對應約 $42.6B）

延續上半週的 IPO 傳聞，OpenAI 本週再出一記震撼彈：Sam Altman 直接向 Trump 總統、商務部長 Lutnick、財政部長 Bessent 提議，把公司約 5% 股份交給美國政府——以近期 $8520 億估值計算，價值約 $426 億美元。

這是 AI 巨頭與政府關係前所未見的操作，牽動出口管制待遇、政府採購、監管立場；與上半週的機密 IPO 籌備（目標 9 月上市）疊在一起看，OpenAI 正同時佈局資本市場與政治靠攏。

3. Anthropic 本週連發：Claude for Teachers、自助 HIPAA、對話中系統訊息、記憶升級

  • Claude for Teachers（07-14）：免費提供全美 K-12 教師，含教學技能庫與對應全 50 州學術標準、實證課綱的直接連結
  • 自助 HIPAA 設定（07-15）：Enterprise / API 組織可自助審閱 BAA、下載實作指南、一鍵啟用，醫療場景落地門檻大幅降低
  • 對話中系統訊息（07-15~16）：Claude API、Bedrock、Google Cloud 上線，支援 Fable 5 / Mythos 5 / Opus 4.8——可在對話中途插入系統指令
  • 記憶升級（07-10 上線，07-16 報導）：以個別分類條目取代每日記憶摘要，提供更豐富的對話脈絡
「對話中系統訊息」對你跑週報這類多階段 pipeline 特別有價值——可在中途調整 agent 行為而不必重開對話。記憶改成分類條目後，跨對話的脈絡連貫性會更好。

4. 🔻 Claude Code 週用量 +50% 加碼已到期（2026-07-13 18:00 PT）

上半週最緊急的項目已成過去式：Anthropic 自 05-13 起提供的 50% 週用量加碼，已於 07-13 美西時間 18:00 到期，週用量回復到加碼前水準。這是一連串「反 Codex」加碼（5/6 五小時額度翻倍、6 月移除尖峰限制）的收尾——背景是 Codex 同等任務 token 消耗量僅約 Claude Code 的 1/4.2。

如果你這週感覺 Claude Code 週額度比上週緊，就是這個原因。目前沒有續期公告，規劃高用量任務時要以回復後的額度為準。

5. OpenAI 產品與硬體動態：統一搜尋、自訂指令擴容、無螢幕智慧音箱

  • ChatGPT 統一搜尋（07-14）：web / iOS / Android 上跨 chats、projects、images、documents 的統一搜尋上線
  • 自訂指令擴容（07-15）：Enterprise / Business / Education 用戶 custom instructions 上限 1,500 → 5,000 字
  • 首款硬體曝光：OpenAI 首款消費裝置將是可移動、無螢幕的智慧音箱，定位家用「AI companion」（Bloomberg 07-14）——與 Apple 商業機密訴訟對照看，裝置端野心正浮上檯面

6. 開源模型：Inkling（Thinking Machines）、Kimi K3（Moonshot）接力

  • Inkling（Thinking Machines，約 07-15）：開放權重通用型 LLM，定位為可與主流封閉模型競爭的 coding 模型，權重可下載並提供 hosted API
  • Kimi K3（Moonshot AI，07-16）：K2.7 之後的新一代，延續 Moonshot 密集的迭代節奏
  • 延續 W28 判斷：GLM-5.2 仍是最推薦的開源 coding 主力，但 Inkling / Kimi K3 值得列入下一輪實測名單

7. Claude Code 平台更新（07-11）

  • Auto mode 在 Bedrock / Vertex AI / Foundry 預設開啟、Bedrock 更新至 Claude Opus 4.8
  • Chrome 整合脫離 preview：直接向 Anthropic 訂閱的所有用戶開放
  • Claude Desktop 登陸 Ubuntu 22.04+ / Debian 12+（beta），x86_64 與 arm64 皆支援
  • Agent view 與 Remote Control 改善，修正凍結、當機、憑證與多個 worktree/plugin bug

8. 補充：上週遺漏的模型發佈（07-08 ~ 07-09）

Grok 4.5（xAI，2026-07-08）、Meta Muse Spark 1.1（2026-07-09）——釋出日期落在 W28 期間但當週未捕捉到，於此補記。

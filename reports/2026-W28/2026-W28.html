🔥 本週最值得注意

1. GPT-5.6 正式全球上線：Sol / Terra / Luna + ChatGPT Work + Codex 併入 ChatGPT 桌面版（2026-07-09）

OpenAI 在 2026-07-09 把原本僅供約 20 家審批機構試用的 GPT-5.6，正式對 ChatGPT、Codex、API 全面開放。

模型定位特點
Sol旗艦agentic 大幅強化，新增 max 推理強度與 Ultra sub-agent 模式
Terra平衡智能與成本平衡，日常工作主力
Luna高量快速效率取向，高流量工作負載

同日兩項配套發佈：

  • ChatGPT Work：內建 agent，跨 App 蒐集脈絡、拆解目標，直接產出 sheets/slides/docs/web apps
  • Codex 併入 ChatGPT 桌面版：macOS/Windows App 整合 Chat + Work + Codex 三合一
GPT-5.6 Terra（$2.50/$15）與 Claude Sonnet 5 標準價（$3/$15）進入直接競爭區間；Ultra 模式與 Codex multi-agent delegation、Claude subagents 走向一致，三大廠這幾週的共同主線是「單一 session 分裂子代理」。

2. Claude Sonnet 5 成為 Pro/Team/Enterprise 預設模型，Claude in Chrome GA，新增 /radio Claude FM

Anthropic 本週把 Sonnet 5 推進為 Pro、Team Standard、Enterprise 訂閱方案的預設模型：Sonnet 定價、頂級 coding 與工具使用能力、原生 1M-token context window、adaptive thinking 預設開啟。

  • Claude in Chrome 正式 GA
  • Subagents 背景執行：預設在背景執行，不再佔用主對話
  • Claude Desktop on Linux：beta 上線
  • Enterprise 模型權限控管：admin 可控管使用者可存取的模型與 effort level
  • Microsoft 365 write tools 擴大：草擬信件、管理行事曆、更新 OneDrive/SharePoint
  • `/radio` 彩蛋：Claude Code 內建指令開啟 "Claude FM"（音樂人策劃的 24/7 lofi 直播），7/3 重新宣傳貼文單篇破 40 萬瀏覽
Pro/Team/Enterprise 訂閱的 Claude Code 這週應已自動切到 Sonnet 5 當預設；background subagents 代表跑長任務時主對話不再被卡住，值得留意行為變化。

3. Gemini 3.5 Pro 持續延遲：距 6/30 GA 目標已跳票 6 週（截至 2026-07-10）

Gemini 3.5 Pro 仍卡在 Vertex AI 企業版有限預覽，距離 6 月 I/O 承諾已晚 5 週，距離原訂 6/30 GA 目標已跳票 6 週，Google 尚未公佈新時間表。同期 Google Gemini 的 7/4 廣告（AI 重現開國元勳起草獨立宣言）被批「不合時宜」。

三大廠中 Google 這波明顯落後，Sonnet 5 GA、GPT-5.6 全面上線後，Gemini 3.5 Pro 的競爭壓力持續加大。

4. GLM-5.2 登頂開源模型：MIT 授權、1M context，SWE-bench Pro 超越 GPT-5.5

Z.ai 的 GLM-5.2（6 月發佈，本週因與 GPT-5.6/Sonnet 5 對比討論再度成為焦點）目前是最強開源權重模型：MIT 授權、context 從 200K 擴充到 1M tokens、GPQA Diamond 91.2%、首個在 SWE-bench Pro 上超越 GPT-5.5 的開放權重模型，Terminal-Bench 2.1 與 Opus 4.8 差距僅幾分。實測案例：完整一個開發 session 成本僅 $0.265。

同期其他開源模型：

  • Kimi K2.7 Code（Moonshot，6/12）：較 K2.6 在 Kimi Code Bench v2 上 +21.8%，HighSpeed 變體推理快約 6 倍
  • MiniMax M3：首個同時具備頂尖 coding、1M context、原生多模態的開放權重模型，SWE-bench Pro 開放權重榜首 59.0%（僅 API，無 Ollama）
  • DeepSeek V4 Pro：code/math 導向，MIT 授權
  • 澄清：Qwen 3.7 Max 是純 API 專有模型，並非開源；開放權重仍是 Qwen 3.6
若你在評估「開源模型能不能取代訂閱制服務」，GLM-5.2 是本季最值得實測的候選——授權寬鬆、成本極低、benchmark 逼近 Opus 4.8。

5. Codex CLI v0.143.0 穩定版（2026-07-08）

  • Remote plugins 預設啟用，更豐富的 catalog、npm marketplace 來源
  • 系統代理支援（macOS/Windows，含 PAC/WPAD）
  • 新增 codex remote-control pair：從執行中的 daemon 產生手動配對碼
  • Amazon Bedrock 新增 GPT-5.6 Sol/Terra/Luna 支援，含 max reasoning effort
  • MCP tools 預設走 tool search；ChatGPT-hosted MCP servers 可用 session authentication
  • 修正 Windows ConPTY 換行/backspace 問題與 sandbox 憑證重試邊界案例

6. EU AI Act：Cybersecurity and AI 行動計畫 + 簡化包即將生效

  • 2026-07-07：歐盟執委會公佈《Cybersecurity and AI 行動計畫》，因應最先進 AI 模型帶來的資安與韌性挑戰
  • AI Act 簡化包已於 6/29 由歐盟理事會最終放行（6/16 歐洲議會已背書），即將刊登官方公報後生效
  • 高風險規則延後適用日期定案：獨立高風險系統 2027-12-02；內嵌於產品的高風險系統 2028-08-02
  • 透明度規則預計 2026-08 生效
若產品有觸及 EU 使用者，高風險義務時程被明確延後，短期合規壓力降低，但透明度義務下月就要生效，是近期優先項目。

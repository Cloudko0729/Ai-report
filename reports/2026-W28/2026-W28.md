## 本週最值得注意

### 1. GPT-5.6 正式全球上線：Sol / Terra / Luna 三層模型 + ChatGPT Work + Codex 併入 ChatGPT 桌面版（2026-07-09）

OpenAI 在 **2026-07-09** 把上週僅供約 20 家審批機構試用的 GPT-5.6，正式對 ChatGPT、Codex、API 全面開放。

**三層模型：**
| 模型 | 定位 | 特點 |
|------|------|------|
| **Sol** | 旗艦 | agentic 能力大幅強化，新增 **max 推理強度**與 **Ultra 模式**（sub-agents 處理複雜任務） |
| **Terra** | 平衡 | 智能與成本平衡，日常工作主力 |
| **Luna** | 高量快速 | 效率取向，高流量工作負載 |

**同日兩項配套發佈：**
- **ChatGPT Work**：內建於 ChatGPT 的 agent，能跨你的各種 App 蒐集脈絡、把目標拆解成步驟，直接產出完成品（sheets、slides、docs、web apps）。
- **Codex 併入 ChatGPT 桌面版**：macOS / Windows 版 ChatGPT 桌面 App 整合 Chat（問答）、Work（研究與產出）、Codex（軟體開發）三合一。

**對你的意義：** GPT-5.6 Terra（$2.50/$15）與 Claude Sonnet 5 標準價（$3/$15）進入直接competition區間；Ultra 模式的 sub-agent 設計與 Codex multi-agent delegation、Claude Agent SDK 的階層式 subagent 走向一致，三大廠這幾週的共同主線是「單一 session 分裂子代理」。

### 2. Claude Sonnet 5 成為 Pro/Team/Enterprise 預設模型，Claude in Chrome GA，新增 /radio Claude FM

延續 W27 的 Sonnet 5 發佈，Anthropic 本週把它推進為 **Pro、Team Standard、Enterprise 訂閱方案的預設模型**：Sonnet 定價、頂級 coding 與工具使用能力、原生 1M-token context window、adaptive thinking 預設開啟。

**同批更新：**
- **Claude in Chrome**：正式 GA。
- **Subagents 背景執行**：預設在背景執行，不再佔用主對話。
- **Claude Desktop on Linux**：beta 上線。
- **Enterprise 模型權限控管**：admin 可控管使用者能存取哪些模型、可用的 effort level。
- **Microsoft 365 write tools 擴大**：可草擬/寄送信件、管理行事曆、更新 OneDrive/SharePoint 檔案。
- **`/radio` 彩蛋**：Claude Code 內建指令，開啟 "Claude FM"——Anthropic 找音樂人策劃的 24/7 lofi YouTube 直播（5/9 上線），7/3 的重新宣傳貼文單篇破 40 萬瀏覽，被戲稱「先給 Claude Code 一個 lofi 電台，才給我們 Opus 5」。

**對你的意義：** 如果你的 Claude Code 是 Pro/Team/Enterprise 訂閱，這週應該已經自動切到 Sonnet 5 當預設；background subagents 代表跑長任務時主對話不會被卡住，值得留意行為變化。

### 3. Gemini 3.5 Pro 持續延遲：距 6/30 GA 目標已跳票 6 週，Google 未給新時間表（截至 2026-07-10）

Gemini 3.5 Pro 仍卡在 Vertex AI 企業版有限預覽，距離 6 月 I/O 承諺的時間已晚 5 週，距離原訂 **6/30 GA 目標**已跳票 **6 週**，Google 尚未公佈新的公開時間表。

同期還有一則插曲：Google Gemini 的 7/4（美國國慶）廣告用 AI 重現開國元勳起草獨立宣言的畫面，被 Bluesky 用戶批「不合時宜」，歷史學者則吐槽廣告其實幾乎沒真的用到 AI。

**對你的意義：** 三大廠中 Google 這波明顯落後，Sonnet 5 GA、GPT-5.6 全面上線之後，Gemini 3.5 Pro 的競爭壓力持續加大，值得繼續追蹤下週是否放出新時間表。

### 4. GLM-5.2 登頂開源模型：MIT 授權、1M context，SWE-bench Pro 超越 GPT-5.5

Z.ai 的 **GLM-5.2**（2026-06 發佈，本週因 GPT-5.6/Sonnet 5 對比討論而再度成為焦點）目前是最強開源權重模型：

- **MIT 授權**、context window 從前代 200K 擴充到 **1M tokens**
- **GPQA Diamond 91.2%**
- **首個在 SWE-bench Pro 上超越 GPT-5.5 的開放權重模型**
- Terminal-Bench 2.1 上與 Claude Opus 4.8 差距僅幾分，且領先 Gemini 3.1 Pro
- 實測案例：完整一個開發 session（規劃、實作、code review、修正）成本僅 **$0.265**

**同期其他開源模型動態：**
- **Kimi K2.7 Code**（Moonshot，6/12）：較 K2.6 在 Kimi Code Bench v2 上 **+21.8%**，另有 HighSpeed 變體推理速度快約 6 倍
- **MiniMax M3**：首個同時具備頂尖 coding、1M context、原生多模態的開放權重模型，SWE-bench Pro 開放權重榜首 59.0%（僅 API，尚無 Ollama 支援）
- **DeepSeek V4 Pro**：code/math 導向，MIT 授權
- 澄清：**Qwen 3.7 Max 是純 API 專有模型**，並非開源；開放權重仍停留在 Qwen 3.6（35B-A3B MoE 4/17、27B dense 4/23，皆 Apache 2.0）

**對你的意義：** 如果你在評估「開源模型能不能取代訂閱制服務」，GLM-5.2 是本季最值得實測的候選——授權寬鬆、成本極低、benchmark 逼近 Opus 4.8。

- **2026-07-07**：歐盟執委會公佈《Cybersecurity and AI 行動計畫》，協調會員國、企業、公部門因應最先進 AI 模型帶來的資安與韌性挑戰。

**對你的意義：** 若產品有觸及 EU 使用者，高風險義務的時程被明確延後，短期合規壓力降低，但透明度義務下月就要生效，是近期優先項目。

## 依你背景的分類

### AI Coding Assistant
- **Codex CLI v0.143.0**：remote plugins、Bedrock GPT-5.6 支援、MCP tool search 預設開啟，值得升級測試
- **Claude subagents 背景執行**：跑週報 pipeline 這類多步驟任務時，可留意主對話是否更順暢
- **ChatGPT 桌面版三合一**：如果你同時用 Codex 和 ChatGPT，現在同一個 App 就能切換 Chat/Work/Codex

### Local LLM
- **GLM-5.2 是本季最值得實測的開源模型**：MIT 授權、1M context、對標 Opus 4.8，成本極低（單一 session 約 $0.265）
- **MiniMax M3 仍卡 API only**：等 Ollama 支援後才能本地跑
- **Ollama v0.31.2**：舊款 NVIDIA GPU 也能用 flash attention，Apple Silicon 上 Gemma 4 速度提升近 90%

### Agent 架構
- **三大廠同步走向「sub-agent 分裂」**：GPT-5.6 Ultra 模式、Claude subagents 背景執行、Codex multi-agent delegation，方向一致
- **ChatGPT Work**：跨 App 蒐集脈絡、拆解目標、直接產出成品，是「agent 取代具體工作流程」趨勢的又一例證

🔥 本週最值得注意

1. Anthropic Agent "Dreaming" 機制

Anthropic 發表 agent dreaming 機制，讓 agent 回顧過去 sessions、找出行為 pattern 並改善後續決策。目前整合進 Claude Managed Agents，sub-agent delegation 進入 public beta。

若你在追蹤 agentic workflow、multi-agent orchestration，這是本週最重要的架構更新。

2. OpenAI Realtime API：三個 Audio Models 上線

  • GPT-Realtime-2：GPT-5 等級 reasoning voice
  • GPT-Realtime-Translate：即時多語言翻譯
  • GPT-Realtime-Whisper：streaming speech-to-text
Whisper 替代方案出現，streaming transcription 工程路徑多一條，值得與現有 faster-whisper pipeline 比較成本與延遲。

3. Claude Code rate limits 提高

Anthropic 提高五小時 rate limits，移除部分 peak-hour 限制，並宣布 SpaceX Colossus 1 compute 合作。

Claude Code 長任務被中斷的頻率應該會降低，agentic coding 可以跑更長的工作。

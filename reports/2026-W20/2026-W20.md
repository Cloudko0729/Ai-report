### 2. OpenAI Realtime API：三個 Audio Models 上線
- `GPT-Realtime-2`：GPT-5 等級 reasoning voice
- `GPT-Realtime-Translate`：即時多語言翻譯

**對你的意義：** Whisper 替代方案出現，streaming transcription 工程路徑多一條，值得與現有 faster-whisper pipeline 比較成本與延遲。

**對你的意義：** Claude Code 長任務被中斷的頻率應該會降低，agentic coding 可以跑更長的工作。

### Anthropic 「Dreaming」與 Managed Agents

**UPDATE: Anthropic's New 'Dreaming' Feature Makes Claude Dangerously Smart**
Anthropic 為 Claude 推出「Dreaming」功能，賦予 AI 長期記憶與自我反思能力。三大重點：(1) Claude 能非同步回顧 100 個對話並清除 Memory rot；(2) 無需重訓基礎模型即可自我提升，法律 AI 任務完成率提升近 6 倍；(3) 主 agent 可將任務拆解並指派子 agent 平行處理。

### Cursor 3 / Agentic Coding

**Cursor 3 Changes Everything for Agentic Coding**
Cursor 3 進化為 agents 的 orchestration layer：支援多模型 multi-agent 並行（Claude / Gemini / OpenAI）、視覺化 Design Mode 直接點選 UI 元素修改、Cloud Agents 可徹夜運行並透過 Slack 遠端指令，所有修改一鍵建立 PR。

**Cursor 3 is Not What You Think**
全視角 Agent 工作區支援跨專案多 agent 同時運行，Composer 2 推論速度快且 API 成本低，支援本地任務一鍵移交雲端執行，Marketplace 擴充 MCP servers 與 Skills。

---

### OpenAI Realtime API

**Introducing gpt-realtime in the API（官方）**
原生 end-to-end 語音模型，強化 Instruction Following 與 Function Calling，首度支援 Image Input，新增 SIP 與 MCP 支援。T-Mobile 導入打造高擬真 AI 客服。

**GPT Realtime 2 Explained**
GPT Realtime 2 內建 GPT-5 等級 Chain of Thought reasoning，延遲 < 400ms，128K context window，GPT Realtime Translate 提供同步翻譯並還原說話者語氣，支援 Parallel Tool Execution。

## 本週最值得注意

### 1. vLLM v0.21.0 — 本週最重量級 inference 更新

vLLM 釋出 v0.21.0，帶來三個核心升級：**KV Offload + Hybrid Memory Allocator**（將 KV cache 卸載至 CPU RAM 或 NVMe，大幅提升長 context 容量）、**Thinking Budget Speculative Decoding**（可在推理品質與速度間動態平衡）、以及 **Transformers v5 遷移**（統一模型載入路徑，降低維護成本）。

**對你的意義：** local LLM serving 的長 context 瓶頸有直接解法；KV Offload 讓有限 VRAM 的機器也能跑更大 context。

---

### 2. Anthropic Opus 4.7 + Claude Design

Anthropic 本週雙管齊下：**Opus 4.7** 新增 Model Router（自動依問題難度切換底層模型，平衡成本與效能），Fast mode 現在 Opus 4.7 驅動，Claude Code 受益；**Claude Design** 是 AI 視覺設計工具（視為 Figma 競爭者），由前 Instagram 創辦人主導。另外 **Claude Platform 正式上線 AWS**，支援 IAM 整合與全套 API。

**對你的意義：** Claude Code 的 Fast mode 更聰明了，Opus 4.7 的自適應思考在非程式任務明顯改善。

---

### 3. GitHub Copilot 全面 Agent 化

本週兩個重大進展：(1) **GitHub Copilot app** 進入 Technical Preview（GitHub-native desktop agentic dev）；(2) **GPT-5.3-Codex 成為 Copilot Business/Enterprise base model**。Copilot coding agent 新增 Agents Panel、預設整合 **Playwright + GitHub MCP servers**，可執行網頁測試與截圖。每次建 PR 只扣一次 premium request，適合清理 tech debt 或預建 70~80% 功能骨架。

**對你的意義：** Copilot 現在可以像 Claude Code 一樣跑 agentic 任務，且已整合 Playwright MCP。

### Anthropic Launches Claude Design and Opus 4.7

### Build Better Agent UX: Streaming Progress with LangChain
技術教學：耗時 tool call 如何用 `config.writer` 向前端發送自訂串流事件（帶類型標籤與 `tool_call_id`），前端用 `use_stream` hook 的 `oncustom` 處理器接收，搭配 TypeScript 型別驗證，可同步渲染多個工具的即時進度卡片。

### GitHub Copilot Agent Mode + Copilot Coding Agent（官方 GitHub Checkout）
Copilot Agent Mode 指派任務給 Copilot Cloud Agent；coding agent 新增 Agents Panel、預設啟用 Playwright + GitHub MCP servers（網頁測試、截圖、讀 issue）。適合 tech debt 清理或 70~80% 初期功能預建，全付費方案開放，每次 PR 扣一次 premium request。未來將推 Public API 並支援自動修復 GitHub Actions 失敗。
